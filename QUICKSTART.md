# 🚀 快速开始指南

欢迎使用智能超链接识别插件！本指南将帮助您在 5 分钟内完成安装和配置。

## 📋 前置要求

-   Google Chrome 浏览器（版本 88 或更高）
-   基本的浏览器操作知识

## ⚡ 快速安装（3 步）

### 步骤 1: 下载项目

```bash
# 克隆仓库
git clone https://github.com/yourusername/SmartHyperlinkRecognition.git

# 或者直接下载 ZIP 文件并解压
```

### 步骤 2: 加载到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions`
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择项目文件夹

### 步骤 3: 开始使用

✅ 安装完成！插件会自动在所有网页上工作。

## 🎯 立即测试

### 测试页面

创建一个简单的 HTML 文件测试效果：

```html
<!DOCTYPE html>
<html>
	<head>
		<title>测试页面</title>
	</head>
	<body>
		<h1>智能超链接识别测试</h1>

		<p>访问我们的网站 www.google.com</p>
		<p>或者发送邮件到 test@example.com</p>
		<p>查看项目 https://github.com/example/project</p>
		<p>本地服务器 localhost:3000</p>

		<h2>这些不会被转换：</h2>
		<pre>代码块中的 www.example.com</pre>
		<a href="#">已有链接 www.test.com</a>
	</body>
</html>
```

保存为 `test.html`，用 Chrome 打开，您会看到纯文本 URL 自动变成了可点击的链接！

## 🎨 基本配置

### 打开设置

1. 点击浏览器工具栏的插件图标 ✍️
2. 点击 **"⚙️ 设置"** 按钮

### 推荐配置

**新手推荐：**

-   ✅ 在新标签页中打开链接
-   黑名单：留空（所有网站都启用）
-   排除标签：使用默认值

**高级用户：**

-   根据需要添加特定网站到黑名单
-   自定义排除的 HTML 标签

## 💡 使用技巧

### 1. 快速开关

点击工具栏图标可以快速启用/禁用插件，无需打开设置页面。

### 2. 网站黑名单

如果某个网站不需要转换功能，添加到黑名单：

```
github.com
stackoverflow.com
```

### 3. 刷新页面

修改设置后，点击 **"🔄 刷新页面"** 按钮使更改生效。

## 🔍 常见使用场景

### 场景 1: 论坛和社区

在 Reddit、知乎等社区网站，用户经常发布纯文本链接，插件会自动转换。

### 场景 2: 技术文档

阅读技术文档时，文档中的 URL 引用会自动变成可点击链接。

### 场景 3: 聊天和评论

在线聊天或评论区的链接分享更加便捷。

### 场景 4: 邮件和联系方式

网页上的邮箱地址会自动变成 mailto 链接。

## 🎓 进阶功能

### 自定义排除标签

默认排除：`SCRIPT, STYLE, NOSCRIPT, TEXTAREA, PRE, CODE, A`

如果需要添加其他标签（如 `BUTTON`）：

```
SCRIPT, STYLE, PRE, CODE, A, BUTTON
```

### 查看统计

在设置页面可以看到总转换次数等统计信息。

## ❓ 遇到问题？

### 插件不工作

1. ✅ 检查插件是否启用（图标显示 "ON"）
2. 🔄 刷新页面
3. 🚫 检查网站是否在黑名单中

### 转换效果不理想

1. ⚙️ 调整排除标签设置
2. 🐛 在 GitHub 提交 Issue
3. 📧 联系开发者

### 性能问题

插件经过优化，对页面性能影响极小。如果遇到问题：

1. 将特定网站添加到黑名单
2. 检查是否与其他扩展冲突
