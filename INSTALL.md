# 智能超链接识别 - 安装指南

## 快速开始

### 方法一：开发者模式安装（推荐用于测试）

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions` 并回车

2. 在页面右上角启用 **"开发者模式"**

3. 点击左上角的 **"加载已解压的扩展程序"** 按钮

4. 选择本项目的根目录文件夹

5. 安装成功！您会看到插件图标出现在浏览器工具栏

### 方法二：打包安装

1. 在 `chrome://extensions` 页面启用开发者模式

2. 点击 **"打包扩展程序"**

3. 选择本项目根目录，点击 **"打包扩展程序"**

4. 生成 `.crx` 文件后，将其拖拽到 Chrome 扩展页面即可安装

## 使用说明

### 首次使用

安装完成后会自动打开欢迎页面，介绍插件的主要功能。

### 日常使用

1. **自动工作:** 插件默认在所有网页上自动运行，无需手动操作

2. **快速开关:** 点击工具栏的插件图标，可以快速启用/禁用功能

3. **高级设置:** 右键点击插件图标 → 选项，可以进行详细配置

### 配置选项

-   **在新标签页打开链接:** 控制转换后的链接是否在新标签页打开
-   **网站黑名单:** 添加不希望插件工作的网站域名
-   **排除标签:** 自定义哪些 HTML 标签内的文本不进行转换

## 测试建议

建议在以下类型的网页上测试插件效果：

1. 论坛和社区网站（如 Reddit、知乎）
2. 技术文档和博客
3. 社交媒体平台
4. 在线聊天和评论区

## 故障排除

### 插件不工作？

1. 检查插件是否已启用（工具栏图标应显示 "ON"）
2. 刷新页面重试
3. 检查当前网站是否在黑名单中
4. 查看浏览器控制台是否有错误信息

### 转换效果不理想？

1. 进入设置页面调整排除标签
2. 检查是否与其他扩展程序冲突
3. 在 GitHub Issues 中反馈具体问题

## 卸载

1. 打开 `chrome://extensions`
2. 找到 "智能超链接识别"
3. 点击 **"移除"** 按钮
