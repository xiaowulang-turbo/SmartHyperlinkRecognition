# 更新日志

## [1.0.4] - 2026-03-28

### ✨ 新增功能

-   🏷️ 全新白名单机制：从全局开关+黑名单模式切换为按站点白名单激活模式
-   🎯 全新 Popup 界面：显示当前域名、按站点开关、转换链接计数统计
-   📊 Badge 实时计数：自动显示当前页面已转换的链接数量
-   🌗 Popup 暗黑模式支持新增的 UI 组件
-   🔢 Popup 底部显示扩展版本号

### 🐛 问题修复

-   修复白名单域名匹配过于宽松的问题：将子字符串匹配改为精确/后缀匹配，避免 `example.com` 误匹配 `badexample.com`
-   修复白名单站点切换标签页时 Badge 旧计数残留的问题
-   修复 Content Script 从不主动上报链接计数导致 Badge 始终为空的问题
-   修复在子域名页面（如 `docs.github.com`）无法通过 Popup 关闭父域名（`github.com`）白名单的问题

### 🔧 技术改进

-   简化 `updateBadgeForTab` 为无条件重置，职责更清晰
-   新增 `sendLinkCount` 防抖机制（300ms），避免高频 DOM 变更时重复发送消息
-   升级时自动迁移旧配置（移除 `enabled`/`blacklist`，补充 `whitelist`）
-   更新 i18n 翻译，适配白名单相关的新文案

## [1.0.3] - 2026-03-08

### ✨ 新增功能

-   🔧 URL 匹配精确度优化，大幅减少误匹配
-   ⚙️ 新增 URL 匹配设置选项（精确模式、文件扩展名排除）
-   🎯 智能排除版本号、统计数据等常见误匹配场景

### 🐛 问题修复

-   修复 `13804.46` 等数字统计被误识别为 URL 的问题
-   修复版本号（如 `1.0.0.290`）被误匹配的问题
-   修复文件扩展名（如 `config.json`）被误识别的问题
-   修复 welcome.html 的 CSP 内联脚本违规问题

### 🔧 技术改进

-   优化 URL 正则表达式，要求协议头或 www 前缀
-   添加多层验证过滤器（数字排除、版本号排除、文件扩展名排除）
-   改进欢迎页面脚本加载方式，符合 CSP 规范

## [1.0.0] - 2025-12-01

### ✨ 新增功能

-   🔗 自动识别并转换网页中的纯文本 URL 为可点击链接
-   📧 支持邮箱地址识别和转换
-   ⚡ 实时监听 DOM 变化，处理动态加载的内容
-   🧠 智能排除规则，避免破坏代码块和已有链接
-   🎨 美观的弹出窗口界面，支持快速开关
-   ⚙️ 完整的设置页面，支持自定义配置
-   🚫 网站黑名单功能
-   🎯 可自定义排除的 HTML 标签
-   🆕 首次安装欢迎页面
-   📊 插件状态徽章显示

### 🎨 界面设计

-   采用蓝色渐变主题设计
-   响应式布局，适配不同屏幕尺寸
-   流畅的动画效果
-   现代化的 UI 组件

### 🔧 技术特性

-   使用 Manifest V3 规范
-   高效的 TreeWalker API 遍历文本节点
-   MutationObserver 监听动态内容
-   Chrome Storage API 持久化配置
-   消息传递机制实现组件通信

### 📝 文档

-   完整的 README.md 说明文档
-   详细的安装指南 (INSTALL.md)
-   MIT 开源许可证
-   代码注释完善

### 🐛 已知问题

-   Shadow DOM 内容暂不支持
-   某些特殊网站可能需要添加到黑名单
