# 更新日志

## [1.0.0] - 2024-12-01

### ✨ 新增功能

-   🔗 自动识别并转换网页中的纯文本 URL 为可点击链接
-   📧 支持邮箱地址识别和转换
-   ⚡ 实时监听 DOM 变化，处理动态加载的内容
-   🧠 智能排除规则，避免破坏代码块和已有链接
-   🎨 美观的弹出窗口界面，支持快速开关
-   ⚙️ 完整的设置页面，支持自定义配置
-   🚫 网站黑名单功能
-   🎯 可自定义排除的 HTML 标签
-   🆕 首次安装欢迎页面
-   📊 插件状态徽章显示

### 🎨 界面设计

-   采用蓝色渐变主题设计
-   响应式布局，适配不同屏幕尺寸
-   流畅的动画效果
-   现代化的 UI 组件

### 🔧 技术特性

-   使用 Manifest V3 规范
-   高效的 TreeWalker API 遍历文本节点
-   MutationObserver 监听动态内容
-   Chrome Storage API 持久化配置
-   消息传递机制实现组件通信

### 📝 文档

-   完整的 README.md 说明文档
-   详细的安装指南 (INSTALL.md)
-   MIT 开源许可证
-   代码注释完善

### 🐛 已知问题

-   Shadow DOM 内容暂不支持
-   某些特殊网站可能需要添加到黑名单
